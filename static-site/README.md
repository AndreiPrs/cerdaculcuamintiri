# Cerdacul cu Amintiri Static Website

This is a static website for a traditional house from Bucovina, replicating the menu and page structure of pensiuneabucovina.ro.

## Structure
- `index.html`: Homepage
- `about.html`: About the house
- `gallery.html`: Photo gallery (auto-loads images from `photos` folder)
- `reservation.html`: Reservation form (uses Formspree, sends to testing@testing.testing)
- `contact.html`: Contact information
- `assets/`: CSS and JS files
- `photos/`: Place your images here; gallery updates automatically

## Reservation Form
- Requires date, number of people, email, and phone (all mandatory)
- Uses Formspree for email submissions
- Update Formspree form ID in `reservation.html` for your email

## Hosting
- Upload all files to GoDaddy hosting (public_html or root directory)
- No backend required

## Customization
- Add new photos to `photos/` folder to update gallery
- Edit content in HTML files as needed
